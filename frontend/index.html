Busca CNPJ Simples API

Consulta de CNPJs simplificada

A Busca CNPJ Simples é uma API pública e open-source para consultas rápidas de CNPJs da Receita Federal. Ideal para integrações e automações em sistemas, sem a necessidade de tokens para a autenticação.

Sobre a API

A Busca CNPJ Simples utiliza a base pública de CNPJs disponibilizada pela Receita Federal, permitindo consultas rápidas e simplificadas de informações cadastrais de empresas, sem a necessidade de lidar com arquivos massivos.

Os dados são atualizados mensalmente, sempre que a Receita Federal libera um novo arquivo — geralmente no início de cada mês. Nosso processo de atualização garante que a API esteja sincronizada em até 3 a 4 dias após a publicação oficial.

Se preferir, você também pode acessar diretamente os arquivos originais da Receita Federal através deste link: Base de Dados - Receita Federal .

Como usar

Faça requisições HTTP utilizando o CNPJ desejado:

Requisição
GET https://api-buscacnpjsimples.com.br/cnpj/00000000000191
Resposta JSON
{
  "cnpj": "00000000000191",
  "razao_social": "EMPRESA EXEMPLO LTDA",
  "nome_fantasia": "EMPRESA EXEMPLO",
  "uf": "SP",
  "municipio": "São Paulo",
  "data_abertura": "1995-07-12"
}

Como a API foi Construída

Pipeline ETL: Pipeline automatizado que extrai dados da Receita Federal, converte para formato Parquet (otimizado para processamento) e carrega em PostgreSQL. Arquitetura modular em Python com componentes especializados: downloader.py, processor.py, parquet_converter.py e database.py.


Stack Tecnológica: FastAPI + Python 3.11, PostgreSQL, Docker e VPS. Documentação automática via Swagger, código versionado no GitHub para total transparência e contribuições open-source.


Processo de Atualização: Atualização mensal automatizada via cron scheduler, com janela de 4 dias após publicação da Receita Federal. O sistema detecta, processa e atualiza os dados sem intervenção manual.
