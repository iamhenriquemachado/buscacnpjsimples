Introdução

A Busca CNPJ Simples é uma API pública e open-source que oferece acesso gratuito às informações de empresas brasileiras. Informe um CNPJ válido e obtenha um JSON pronto para uso. Você terá acesso a um único endpoint.

Documentação

A API utiliza o método HTTP GET e pode ser consultada por meio do seguinte endpoint principal, retornando os dados em formato JSON estruturado, ideal para consumo em aplicações modernas. Para começar a utilizar a API, utilize o seguinte endpoint:

Requisição

Base URL: https://api-buscacnpjsimples.com.br/cnpj/:cnpj

Parâmetros

Formato Exemplo Descrição
Somente números 00000000000191 Formato mais simples e recomendado para chamadas programáticas.
Com pontuação completa 00.000.000/0000-91 Inclui pontos, barra e hífen. Aceito pela API sem problemas.
Com pontos e barra (sem hífen) 00.000.000/000091 Alternativa intermediária, útil quando não se deseja usar o hífen.
